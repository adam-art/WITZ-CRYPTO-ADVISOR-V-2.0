01Pengantar

WITZ adalah aplikasi manajemen portofolio kripto generasi berikutnya yang menggabungkan data pasar real-time dari CoinGecko dengan kecerdasan buatan dari 4 provider AI terdepan: Anthropic Claude, OpenAI GPT-4, Google Gemini, dan Groq Llama.

📊
Harga Real-Time
Data diperbarui otomatis setiap 30 detik dari CoinGecko tanpa API key tambahan.
🤖
4 AI Terintegrasi
Pilih dari Claude, GPT-4, Gemini, atau Llama — semua mendukung konteks portofolio live.
🔒
Keamanan Wallet
Aplikasi sepenuhnya terkunci tanpa koneksi wallet. Akses read-only, tidak ada akses kunci privat.
⚠️
Analisis Risiko DeFi
Matriks risiko protokol DeFi lengkap dengan AI scan berbasis portofolio nyata Anda.
📈
Analitik Lengkap
Chart 30 hari per aset, performa komparatif, dan histori performa portofolio 90 hari.
🌐
Market Intelligence
Dominasi BTC, total market cap, trending coin, dan data volume 24 jam seluruh pasar.
⚠️

Perhatian: WITZ adalah alat edukasi dan analisis. Informasi yang disajikan bukan merupakan saran investasi atau keuangan resmi. Selalu lakukan riset mandiri sebelum mengambil keputusan finansial.

cp .env.example .env.local

copy .env.example .env.local

Jalankan server pengembangan

npm run dev

Buka browser dan kunjungi http://localhost:5173

03Konfigurasi File .env

WITZ menggunakan file .env.local untuk menyimpan semua kunci API dengan aman. API key tidak pernah disimpan di dalam kode aplikasi — semuanya dibaca dari environment variable saat aplikasi berjalan.

🚨

Keamanan Penting: Jangan pernah mengunggah file .env.local ke GitHub atau berbagi isinya. File ini sudah tercatat di .gitignore secara otomatis.

Struktur File .env.local

# ── AI PROVIDERS (pilih minimal 1) ──────────────────────
VITE_ANTHROPIC_API_KEY=sk-ant-api03-xxxxx    # Claude
VITE_OPENAI_API_KEY=sk-xxxxx               # GPT-4
VITE_GEMINI_API_KEY=AIza-xxxxx            # Gemini
VITE_GROQ_API_KEY=gsk_xxxxx              # Llama (Groq)

# ── PASAR (Tidak perlu API key) ──────────────────────────
# CoinGecko Free bekerja otomatis tanpa konfigurasi

# ── DATABASE (Opsional) ───────────────────────────────────
VITE_SUPABASE_URL=https://xxxxx.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGci...

Cara Kerja Proxy Keamanan

WITZ menggunakan Vite proxy yang berjalan di server lokal untuk mengirimkan API key. Ini berarti:

  • API key tidak pernah terekspos di browser atau network tab DevTools
  • Permintaan ke AI dikirim melalui server lokal port 5173
  • Di production, gunakan backend server (Node.js/FastAPI) sebagai proxy

05Dashboard

Dashboard adalah halaman utama WITZ yang memberikan gambaran lengkap kondisi portofolio Anda dalam satu tampilan.

Kartu Statistik Utama

Empat kartu di baris atas menampilkan:

13Keamanan & Privasi

WITZ dirancang dengan prinsip keamanan berlapis. Berikut adalah semua lapisan perlindungan yang aktif:

🔒
Wallet Gate (Gerbang Akses)
100% fitur terkunci tanpa wallet terhubung. Tidak ada data yang dimuat, tidak ada API yang dipanggil sebelum autentikasi wallet.
🔑
API Key di .env (Tidak di Kode)
Semua API key disimpan di file .env.local yang tidak pernah dicommit ke Git dan tidak pernah muncul di kode sumber aplikasi.
🛡
Proxy Vite (Key Tersembunyi)
Permintaan ke AI dikirim melalui proxy lokal. API key tidak pernah terekspos di browser DevTools atau network tab.
🧹
Sanitasi Input
Semua input pengguna dibersihkan dari tag HTML dan karakter berbahaya sebelum dikirim ke AI (perlindungan XSS).
👁
Read-Only Wallet
WITZ hanya meminta izin baca alamat wallet. Tidak ada izin tanda tangan, transfer, atau interaksi kontrak apapun.
🎭
Masking Alamat
Alamat wallet selalu ditampilkan dalam format tersamarkan (0x8f4a····2d1e) untuk melindungi privasi Anda.
⏱
Rate Limiting
Pembatas laju panggilan API bawaan mencegah penggunaan berlebihan yang tidak disengaja dan melindungi dari serangan brute force.
💾
Tidak Ada Data Tersimpan
WITZ tidak menyimpan data portofolio Anda di server manapun. Semua data hanya ada di memori browser selama sesi berlangsung.

Rekomendasi Keamanan untuk Pengguna

  • Selalu periksa alamat URL sebelum menghubungkan wallet
  • Jangan pernah menyetujui permintaan tanda tangan yang tidak Anda pahami
  • Gunakan wallet terpisah untuk eksperimen — jangan gunakan wallet utama Anda
  • Pertimbangkan hardware wallet (Ledger) untuk aset dalam jumlah besar
  • Jangan bagikan file .env.local kepada siapapun

15Pertanyaan Umum (FAQ)

Apakah WITZ aman digunakan dengan wallet asli saya?
Ya, WITZ hanya meminta akses baca (read-only) ke alamat wallet Anda. Aplikasi tidak pernah meminta izin tanda tangan, transfer, atau interaksi smart contract. Namun untuk keamanan maksimal, kami tetap menyarankan menggunakan wallet terpisah saat melakukan testing aplikasi baru.
Data portofolio saya tersimpan di mana?
Tidak disimpan di mana pun. Semua data hanya ada di memori browser selama sesi berlangsung. Ketika Anda menutup tab atau memutus koneksi wallet, semua data terhapus otomatis. WITZ tidak memiliki backend server yang menyimpan data pengguna.
Mengapa saya perlu menghubungkan wallet terlebih dahulu?
Koneksi wallet adalah mekanisme autentikasi utama WITZ. Ini memastikan hanya pengguna yang memiliki akses ke wallet yang dapat melihat data portofolio. Tanpa wallet gate, siapapun yang mengakses URL aplikasi bisa melihat data.
Provider AI mana yang paling bagus untuk pemula?
Kami merekomendasikan memulai dengan Groq (Llama 3.3) karena sepenuhnya gratis tanpa kartu kredit, respons sangat cepat, dan tidak ada batas yang ketat. Setelah terbiasa, upgrade ke Claude (Anthropic) untuk analisis yang lebih mendalam dan nuanced.
Apakah data harga dari CoinGecko akurat?
Ya, CoinGecko adalah salah satu sumber data kripto terpercaya di industri. Data diperbarui setiap 30 detik. Namun selalu ada potensi keterlambatan (delay) beberapa detik hingga menit dibandingkan data exchange real-time.
Bisakah saya menambah aset kripto lain ke portofolio?
Dalam versi saat ini, daftar aset dikonfigurasi di file src/services/market.js. Tambahkan ID CoinGecko aset baru ke objek COIN_IDS dan jumlah kepemilikan ke HOLDINGS.
Bagaimana cara deploy WITZ ke internet (production)?
Jalankan npm run build untuk membuat folder dist. Upload ke Vercel, Netlify, atau VPS. PENTING: Di production, jangan gunakan proxy Vite — buat backend API server tersendiri (Node.js/FastAPI) untuk mengelola API key dengan aman.
Kenapa WITZ AI menjawab dalam Bahasa Indonesia?
System prompt WITZ dirancang untuk meminta AI menjawab dalam Bahasa Indonesia. Anda tetap bisa bertanya dalam Bahasa Inggris dan AI akan menyesuaikan.
